# MARC export

## Formats

- `/books/export/marc.mrc` - ISO 2709 binary (`application/marc`)
- `/books/export/marc.xml` - MARCXML (`application/marcxml+xml`)
- `/books/export/marc.mrk` - MARCMaker text (`text/plain`)

below is only added when its source data is present, except 001/003 which are

### 952 subfields (Koha item holdings)

Only emitted for authenticated exports (`include_holdings=True`); locations
flagged `non_public` (or with a `non_public` ancestor) are skipped entirely
unless the requesting user has permission to see them, in which case `$5 1`
is added to mark the item restricted in Koha.

| Subfield | Meaning | Source |
|---|---|---|
| `$a` | home library | root ancestor ("building") location name |
| `$b` | holding library | same as `$a` |
| `$c` | shelving location | `Location.full_path()` of the item's location |
| `$p` | barcode | entry number |
| `$y` | item type | `"BK"` (fixed) |
| `$o` | call number | `BookEntry.skoufas_classification`, if set |
| `$z` | copies note | `"<n> copies"`, if `copies > 1` |
| `$z` | donor note | `"Donated by: <names>"`, if this entry number has its own donors |
| `$5` | restricted | `"1"`, only if the location is non-public |

If the entry number has no location at all, only `$p` and `$y` are emitted.

### 856 subfields (linked images)

| Subfield | Meaning | Source |
|---|---|---|
| `$u` | absolute URL | e.g. `https://yourdomain.com/media/book_images/1038/cover.jpg` |
| `$3` | label | `"Cover image"` / `"Back cover image"` / `"Internal image"` / `"Image"` |
| `$q` | MIME type | `"image/jpeg"` (fixed) |
| `$z` | caption | `BookEntryImage.caption`, if set |
